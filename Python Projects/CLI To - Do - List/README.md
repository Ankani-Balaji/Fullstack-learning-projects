# 🧠 CLI To-Do List

A simple and efficient **Command Line Interface (CLI) To-Do List application** built using Python.

---

## 🚀 Features

* ➕ Add Tasks
* 📋 View Tasks
* ✅ Mark Tasks as Done
* ❌ Delete Tasks with confirmation
* 🔍 Search Tasks (case-insensitive)
* 🚫 Prevent Duplicate Tasks using Set
* ⚡ Priority Support (High / Low)

---

## 🛠️ Tech Stack

* Python
* Lists, Sets, Dictionaries
* Functions & Modular Programming
* String Manipulation
